
# PDF Maker - Intelligent Document Suite

A professional, AI-powered suite for document and image processing.

## Features
- **Image-to-PDF**: Bundle images into high-quality PDF documents using jsPDF.
- **Smart Merger**: Combine multiple PDFs seamlessly with pdf-lib.
- **Smart Compression**: Browser-based image size reduction.
- **Precision Resizer**: Pixel-perfect image scaling using browser canvas.
- **PDF Decomposition**: Extract pages from PDF documents into a ZIP archive.

Built with Next.js, Genkit, and Firebase.
