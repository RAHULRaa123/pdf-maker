# **App Name**: DocuPix

## Core Features:

- Image-to-PDF Conversion: Quickly bundle multiple images into a single, high-quality PDF document.
- Smart Merger: Combine disparate PDF files into one organized document with an intuitive drag-and-drop interface.
- Intelligent Compression Tool: An AI-powered tool that analyzes visual fidelity to find the perfect balance between file size reduction and image clarity.
- Precision Resizer: Scale and crop images to specific dimensions or aspect ratios for any platform requirement.
- PDF Decomposition: Convert complex PDF pages back into individual, high-resolution image formats like JPG or PNG.
- Universal Share & Save: Directly export processed files to system storage or share them through standard mobile share sheets.

## Style Guidelines:

- Primary color: Deep Technical Blue (#2E6DA4) chosen for its association with precision and productivity. Background: Clean Paper White (#F7F9FA) with a subtle cool tint to enhance readability. Accent: Vibrant Royal Purple (#6B7CFF) for clear, actionable call-to-action buttons.
- Using 'Inter' for its objective, machined look that remains legible even in high-density utility apps. Its neutral character provides a modern, tool-like aesthetic perfect for productivity software.
- Minimalist home screen featuring large, distinctive cards for each tool, utilizing a grid system that adapts seamlessly from phone to tablet orientations.
- Geometric line-art icons that use uniform stroke weights to maintain a clean and professional appearance throughout the interface.
- Smooth micro-interactions such as button depth changes and layout shifts during file processing to provide immediate visual feedback to the user.
